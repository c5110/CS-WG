✨ 核心功能
🏷️ 标记功能
选中网页中的文本

从菜单中选择状态：入库/淘汰/等待/喜欢/讨厌

页面中自动显示彩色标签

选中后在右键油猴里面找到脚本，自己研究

油猴脚本新建或者导入，然后开服务端，然后刷新网页就有了。
